This is Conway's Game of Life.

It is a celullar automaton created by John Horton Conway in 1970.

It is very simple. We have a grid in which each square can be either "dead" or "alive".

In order to determine that, you must run each square through a set of rules and set their state.

These rules are based on each square's "neighbors": the 8 other squares surrounding them.

After running the rules through the entire grid, a generation is created.

The game plays infinitely by creating generation after generation.

The rules are as follows:

  • Any "live" cell with 2 or 3 "live" neighbors stays "live"
  • Any "dead" cell with exactly 3 "live" neighbors becomes "live"
  • Any other cells become "dead"

While the game's rules are very simple, it can create some very complex shapes and patterns.

The grid on the left is interactive. You can draw on it by turning on Pause and using your mouse clicks. You can also pause by tapping the Spacebar.

Try recreating some of the shapes in the game's Wikipedia page, like Gosper's Glider Gun.
